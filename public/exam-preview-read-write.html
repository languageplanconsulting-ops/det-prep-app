หลัก UX ที่ใช้ — Krug + Cagan:

  • Thai-first — ภาษาไทยตัวใหญ่/หนา · ภาษาอังกฤษ italic เล็ก (เพราะ user อ่านไทยก่อน, อังกฤษเป็น reference)
  • ชุดข้อสอบ แทน Round — "Round" ฟังดูเหมือนเกม "ชุดข้อสอบ" คนไทยเข้าใจทันทีว่าเป็นข้อสอบ
  • หัวข้อ แทน topic — คำไทยที่นักเรียนคุ้นเคย
  • ปุ่ม "▶ เริ่มเขียน" แทน "START EXAM" — กดแล้วไม่กดดัน เป็นการเริ่มกิจกรรม ไม่ใช่เริ่มสอบ
  • เห็นคะแนนเดิมบน tile — "✓ ทำแล้ว · 138/160" + ปุ่ม "↻ ทำใหม่" ให้ทำซ้ำได้ทันที
  • Tips from P'Doy speech bubble — ทำให้ instruction รู้สึกเหมือนครูบอก ไม่ใช่ระบบเตือน
  • Filter chips (ทั้งหมด/ยังไม่ทำ/ทำแล้ว) — Krug pattern คุ้นเคย + ลด choice paralysis
  • Breadcrumb — "ฝึก › Production › Read & Write › ชุดที่ 1" บอก "ฉันอยู่ตรงไหน" ตลอด
  • Empty rounds (4, 5) dimmed + "เร็วๆ นี้" — set expectation ดีกว่าซ่อนหรือล็อก
  • คำชี้แจง popup (หน้า 1) ต้องสั้น scan ได้ใน 5 วินาที — ปุ่ม "เลือกชุดข้อสอบ" ใหญ่อยู่ล่างสุด
