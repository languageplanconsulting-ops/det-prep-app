หน้า Landing ใหม่ — 5 แนวทาง

แต่ละเวอร์ชันออกแบบจากหลัก UX/UI psychology คนละข้อ แต่ทุกเวอร์ชันยึดหลักเดียวกัน: ปุ่มหลักชัดเจนปุ่มเดียว · ภาษาไทยนำ · ลดความกังวลของผู้เรียน · ไม่ขายของ — เริ่มจากคุณค่าและการแก้ปัญหา · ออกแบบ mobile-first. ทั้งหมดยังเป็น preview เท่านั้น — ผู้ใช้จริงยังเห็นหน้าเดิม จนกว่าคุณจะเลือก.

หลักการที่ใช้ / Research applied

  • Don't Make Me Think (Krug): หน้าเดิมมีปุ่มแข่งกัน 7+ ปุ่มใน hero (ผิดหลัก Hick's Law) — ทุกเวอร์ชันใหม่เหลือปุ่มหลักเดียว
  • Feeling understood > features: เปิดด้วย "ปัญหาจริง" ของผู้เรียนไทย ไม่ใช่ลิสต์ฟีเจอร์
  • ESL psychology (self-efficacy + anxiety): ตัวบล็อกใหญ่สุดคือความกังวลเรื่องพูด/เขียน — เน้น ฟรี · ส่วนตัว · ไม่ตัดสิน · เริ่มทีละนิด · เห็นชัดว่าต้องแก้อะไร
  • Thai localization: mobile-first, ใช้ไอคอน/ภาพนำ (ภาษาไทยไม่มีเว้นวรรค), PromptPay, ครูคนจริง, รีวิวคนไทย before→after
  • Value-first, ไม่ salesy: ดันดีไซน์ "เช็กระดับฟรี" ขึ้นก่อน — ราคาอยู่ท้าย ๆ และเบา
  • Fogg / goal-gradient: ก้าวแรกเล็กมาก + เห็นเส้นทางสู่เป้าหมายชัด
★ ล่าสุด

Calm Clarity — refined (เวอร์ชันที่คุณเลือก + ปรับตาม feedback)

ต่อยอดจาก Calm Clarity: บอกชัดว่านี่คืออะไร + ประโยชน์ของแต่ละอย่าง (เช็กระดับ · Mock test ที่โชว์จุดอ่อน · ฟีดแบ็กรายข้อแบบ V3 · มินิเลสซัน) · รายการข้อสอบครบทุกแบบ อธิบายตรง ๆ ว่าแต่ละแบบแก้ปัญหาอะไร · ตัดคำว่า "AI" ออก · ไม่มีส่วนโปรโมตตัวผู้สอน (ลดความกดดัน) · มีส่วนคอร์ส → VIP 6 เดือน.

หลัก: Clarity + Gen-Z authenticity + selling-as-a-real-teacher →

5 แนวทางแรก (สำหรับเทียบ)

V1

Calm Clarity

หัวข้อเดียว ปุ่มเดียว เว้นที่ว่างเยอะ — ลดภาระการคิด (cognitive fluency) ให้รู้สึก "โล่ง ไม่กดดัน". ก้าวแรกเล็กที่สุดเท่าที่จะเป็นไปได้.

หลัก: Don't Make Me Think + Fogg →
V2

Empathy-first / เข้าใจหัวอกเด็กไทย

เปิดด้วยความรู้สึกจริง: "อ่านพอได้ ฟังพอรู้เรื่อง แต่พอต้องพูด/เขียนคือตัน". สะท้อน → ปลอบ → ชวนเริ่มทีละนิด. โทนพี่เลี้ยงใจดี ไม่ใช่นักขาย.

หลัก: Emotional resonance →
V3

Proof & Self-Efficacy / เห็นผลก่อนจ่าย

โชว์ feedback จริงกลางหน้า — คะแนน Grammar/Flow + โน้ตภาษาไทยบอกว่าต้องแก้อะไร. "ลองดูก่อนว่ามันช่วยยังไง แล้วค่อยตัดสินใจ".

หลัก: Show-don't-tell + confidence →
V4

The Path / เส้นทางสู่เป้าหมาย

เลือกเป้าคะแนน (105–115 / 116–124 / 125+) แล้วเห็น roadmap เป็นขั้น ๆ จากเช็กระดับ → ฝึกตรงจุด → mock → ถึงเป้า. เปลี่ยน "ภูเขา" ให้เป็นก้าวเล็ก ๆ.

หลัก: Goal-gradient →
V5

Thai-first Mobile

ออกแบบจากพฤติกรรมคนไทยจริง: มือถือก่อน, ไอคอนนำ, บรรทัดสั้น, ปุ่มใหญ่ติดล่างจอแบบแอป, ครู "พี่ดอย" ทักทายลดความเกร็ง, จ่ายผ่าน PromptPay. ลองเปิดบนมือถือ (หรือย่อจอ) จะเห็นชัดสุด.

หลัก: Thai localization + mobile-first →

เลือกตัวที่ชอบที่สุด (หรือผสมจุดเด่นหลายตัว) แล้วบอกผม เดี๋ยวเอาไปทำเป็นหน้า landing จริงใน LandingPageClient.tsx
