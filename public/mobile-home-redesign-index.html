หน้าหลัก — จัดสี/ดีไซน์/เลย์เอาต์ใหม่ 🎨

3 แนวทางแก้ให้หน้า หน้าหลัก ดูดีขึ้น + สอดมาสคอตให้ลงตัว (ไม่ใช่แปะเพิ่ม)

ปัญหาที่เห็นในเวอร์ชันก่อน: สีชนกันเยอะ (cream + การ์ดกรมท่า + การ์ดฟ้าสด + พาสเทล 4 สี + เหลือง) ไม่มีลำดับความสำคัญ · การ์ดใหญ่เต็มกว้างซ้อนกันหลายใบ ดูหนัก · มาสคอตเหมือนถูกแปะเข้าไป ทั้ง 3 แบบนี้ คุมจานสีให้แคบลง + จัดลำดับ + รวมการ์ดฮีโรให้เหลือใบเดียว
VERSION 1

Cohesive Blue แนะนำ

รวม ทักทาย + สตรีค + เป้า + ปุ่มแบบฝึกประจำวัน ไว้ในฮีโรน้ำเงินใบเดียว (มาสคอตอยู่บนนั้น) · Mini Test เป็นแถวบาง · หมวดเป็นการ์ดขาวที่ใส่สีแค่ในไอคอน — สีไม่ตีกัน

  • คุมสีให้เหลือ น้ำเงิน+เหลือง+ขาว/ครีม
  • ลำดับชัด: ฮีโร 1 ใบ → ทดสอบ → หมวด → Mock
  • สะอาด ดู "ออกแบบมา" · พอร์ตง่าย
VERSION 2

Warm Minimal

โทนอุ่น พรีเมียม เว้นวรรคเยอะ ตัวอักษรใหญ่ · ฮีโรการ์ดครีมกับมาสคอตตัวใหญ่ · สถิติเป็นแถบบาง ๆ · หมวดเป็นลิสต์อ่านง่าย ใช้สีน้อยมาก (ครีม/แซนด์/ส้มอิฐจุดเดียว)

  • ดูแพง สงบ ไม่รก
  • เน้นตัวอักษร/ช่องไฟแทนสีเยอะ ๆ
  • มาสคอตเป็นพระเอกในฮีโร
VERSION 3

Bold Playful

สดใสแต่คุมโทน — บล็อกฮีโรน้ำเงินเต็มหัว (มาสคอต+ปุ่มอยู่บนนั้น) แล้วชิปสถิติลอยทับ · ใช้สีหลัก น้ำเงิน+เหลือง + ส้มคอรัลเป็นแอ็กเซนต์เดียว · หมวดชังกี้สีสม่ำเสมอ

  • สดใส เอนเกจ แต่ไม่มั่ว (จานสีแคบ)
  • แบรนด์เด่น (หัวน้ำเงินเต็ม)
  • เหมาะกลุ่มนักเรียน/วัยรุ่น

เหมือนกันทั้ง 3 แบบ

• คงฟีเจอร์จริงของ HomeScreen (สตรีค/XP, แบบฝึกประจำวัน, Mini Test, 4 หมวด, Mock)  •  จานสีแคบลง มีลำดับความสำคัญชัด  •  รวมฮีโรให้เหลือใบเดียว (ไม่ใช่การ์ดใหญ่ซ้อนกันหลายใบ)  •  มาสคอตอยู่ในฮีโร/หัว อย่างกลมกลืน  •  ธีมมือถือจริง + ฟอนต์ Prompt
