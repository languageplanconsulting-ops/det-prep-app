Dictation audio files (local HTML5 playback — no API)

Naming (matches code in src/lib/dictationData.ts):
  easy-1.mp3 … easy-20.mp3
  medium-1.mp3 … medium-20.mp3
  hard-1.mp3 … hard-20.mp3

Place each MP3 in this folder: public/audio/dictation/

Until files exist, the player will show an error banner — that is expected.
You can copy one short clip to all filenames as a temporary placeholder while you record the real tracks.
