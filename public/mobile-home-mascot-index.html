มาสคอตบน "หน้าหลัก" (แอปมือถือ) 📱

3 วิธีเอามาสคอตมาเป็นหน้าตาของแอปบนหน้า หน้าหลัก (HomeScreen) — คงเลย์เอาต์จริง (สตรีค/XP, แบบฝึกประจำวัน, Mini Test, หมวดแบบฝึก, Mock) แล้วสอดมาสคอตเข้าไปคนละแบบ

📱 ต้นฉบับที่แก้: HomeScreen.tsx (แท็บ หน้าหลัก) · มาสคอต = public/mascot-doy.png (พื้นหลังโปร่งแล้ว) · ธีมมือถือจริง (cream + EP blue/yellow + ฟอนต์ Prompt)
👋
VERSION 1

Greeter (ทักทาย) ปลอดภัยสุด

มาสคอตอยู่ในการ์ดทักทายใต้หัวข้อ — มีลูกโป่งคำพูด "วันนี้ฝึกอะไรดี?" แล้วตามด้วยสตรีค/เป้ารายวัน ที่เหลือคงเดิมทั้งหมด

  • เปลี่ยนน้อย เสี่ยงน้อย พอร์ตง่ายสุด
  • อบอุ่น เป็นมิตร ไม่รก
  • มาสคอตโผล่จุดเดียว เด่นชัด
🧑‍🏫
VERSION 2

Coach (โค้ชเช็กอิน)

แบนเนอร์โค้ช "น้องแพลน" เด่นบนสุด — พูดเจาะจงรายวัน ("วันนี้เก็บ Writing ต่อนะ 🔥") พร้อมสตรีค/XP/เป้า + ปุ่มเริ่มฝึก มาสคอตอยู่ในโลโก้ด้วย

  • มีบุคลิก/เสียงชัด เป็นตัวละครประจำแอป
  • กระตุ้นให้กลับมาทุกวัน (daily check-in)
  • ผูกกับระบบ XP/สตรีคที่มีอยู่แล้ว
✨
VERSION 3

Playful (โผล่หลายจุด)

มาสคอตเป็นตัวการ์ตูนสนุก ๆ ที่โผล่ทั่วหน้า — ทักในหัวข้อ, โผล่ออกมาจากการ์ดแบบฝึกประจำวันพร้อมสติกเกอร์, แอบโผล่ข้างหัวข้อหมวด สดใสวัยรุ่น

  • สะดุดตา จำง่าย เอนเกจสูง
  • เหมาะกลุ่มนักเรียน/วัยรุ่น
  • เสี่ยงดู "เล่น" เกินถ้าใช้เยอะ · ต้องคุมพอดี

เหมือนกันทั้ง 3 แบบ

• คงเลย์เอาต์ + ฟีเจอร์จริงของ HomeScreen ทุกอย่าง (สตรีค/XP, แบบฝึกประจำวัน, Mini Test, 4 หมวด, Mock)  •  ใช้ มาสคอตตัวเดียวกัน (พื้นหลังโปร่ง)  •  ธีมมือถือจริง + ฟอนต์ Prompt

ต่างกันที่ บทบาทมาสคอต: V1 คนทักทาย (นิ่ง) · V2 โค้ชที่พูดนำ (มีเสียง เช็กอินรายวัน) · V3 ตัวการ์ตูนสนุก (สติกเกอร์ โผล่หลายจุด)

หมายเหตุ: ชื่อ "น้องแพลน" ใน V2 ตั้งชั่วคราว บอกได้ว่าอยากใช้ชื่ออะไร  ·  มาสคอตปัจจุบันเป็นภาพครึ่งตัว/หน้า ถ้าจะใช้จริงควรมี เต็มตัว + หลายท่าทาง (ยิ้ม/ชูนิ้ว/ดีใจ) เลือกเวอร์ชันที่ชอบแล้วผมพอร์ตลง HomeScreen.tsx จริงให้ (verify บน Expo web)
