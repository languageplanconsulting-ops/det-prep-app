เปลี่ยน "ตามคำบอก" ให้เป็นเส้นทางเกม 🎮

แทนที่จะเป็นแค่รายการชุดข้อสอบเรียงกัน ทั้ง 3 แบบทำให้ผู้เรียนเห็น "เส้นทาง" ที่ชัดเจน — ตัวเองอยู่ตรงไหน, ผ่านมากี่ด่าน, เหลืออีกกี่ด่าน, และมีรางวัลรออยู่ข้างหน้า เพื่อกระตุ้นให้เล่นต่อ

📱 ต้นฉบับที่แก้: ExamPractice (มือถือ) — แท็บระดับ ง่าย/ปานกลาง/ยาก + รายการ "รอบที่ 1 · ชุด N" ธรรมดา · ใช้ข้อมูลจริง (สถานะ ยังไม่ทำ / คะแนน% / ✓ เต็ม)
🛤️
VERSION 1

เส้นทางผจญภัย

เส้นทางคดเคี้ยวแนวตั้งแบบ Duolingo — แต่ละด่านเป็นวงกลม มีตัวละคร "คุณอยู่ตรงนี้" กล่องรางวัลตามทาง และหมุดบอกตำแหน่ง

  • สนุก เห็นความก้าวหน้าเป็นภาพชัดที่สุด
  • ระดับ = "โลก/บท" ที่ปลดล็อกทีละขั้น
  • เหมาะกับกลุ่มที่ชอบเกม · เสี่ยงดู "เด็ก" นิดหน่อย
🗺️
VERSION 2

แผนที่ผจญภัย

แผนที่เป็นภาพ — แต่ละระดับคือ "โลก" (ทุ่งหญ้า/ทะเลทราย) มีก้อนหินเดินทาง ตัวฮีโร่ ต้นไม้ เมฆ และธงรางวัล

  • ดูพรีเมียม มีเอกลักษณ์ น่าจดจำ
  • สื่อ "การเดินทางข้ามระดับ" ได้ดี
  • งานภาพเยอะ · ใช้เวลาทำ/ปรับมากกว่าอันอื่น
🧭
VERSION 3

เส้นเวลาเควส แนะนำ

ไทม์ไลน์แนวตั้ง — การ์ด "ทำต่อ" ใหญ่ด้านบน + สถิติ (ผ่าน/เฉลี่ย/สตรีค) + ด่านเป็นจุดต่อเส้น จัดเป็น "บท" มีแถบรางวัลคั่น

  • ยังอ่านง่าย/สแกนเร็ว แบบรายการเดิม แต่เป็นเกม
  • ทำเป็น React Native ได้เร็วสุด (ไม่มี SVG ซับซ้อน)
  • ต่อยอดใช้กับทุกวิชาได้ทันที · ปลอดภัยที่สุด

เหมือนกันทั้ง 3 แบบ (แก่นของการออกแบบ)

• เห็น "คุณอยู่ตรงไหน" และ "เหลืออีกกี่ด่าน" ทันที  •  แสดง คะแนน% / ✓ เต็ม / REDEEM จากข้อมูลจริง  •  ระดับ (ง่าย→ยาก) กลายเป็น บท/โลกที่ปลดล็อกต่อกัน  •  มี สตรีค 🔥 + รางวัล/หมุดหมาย ตามทาง เชื่อมกับระบบ XP ที่มีอยู่แล้ว (gamification.ts)  •  ปุ่ม "ทำต่อ" เด่นชัดปุ่มเดียว ไม่ต้องเลือกเอง
