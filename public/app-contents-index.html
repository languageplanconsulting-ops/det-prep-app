หัวข้อ “ในแอปมีอะไรให้ใช้บ้าง” — 3 เวอร์ชัน

ต่อยอดจาก 4 เครื่องมือหลัก (เช็กระดับ · Mock · ฟีดแบ็กรายข้อ · เทคนิคสั้นๆ) แล้วเพิ่มคลังข้อสอบครบ 4 ทักษะ (ฟัง · อ่าน · เขียน · พูด) พร้อมไฮไลต์ พูดแบบโต้ตอบ · การอ่านจับใจความ · คำศัพท์. สื่อถึงจุดที่นักเรียนชอบจริง (ลิลลี่: “หาข้อสอบจริงมาฝึกยาก เว็บอื่นอธิบายอังกฤษล้วน”). ไม่มีคำว่า “AI”.

V1

Tabbed skill bank

แท็บ 4 ทักษะ (ฟัง/อ่าน/เขียน/พูด) — กดสลับดูแบบฝึกในแต่ละทักษะ เรียบ สแกนง่าย ไฮไลต์ตัวเด่นด้วยป้าย ⭐.

หลัก: Designing Interfaces (tabs) + Don't Make Me Think →
V2

Visual grid (magazine)

การ์ดไฮไลต์ใหญ่ 3 ใบ (พูดโต้ตอบ · การอ่าน · คำศัพท์) + กริดรวมทุกแบบฝึกแยกตามทักษะ เน้นภาพ/ปริมาณ ให้รู้สึกว่า “คลังเยอะจริง”.

หลัก: Thai visual-first + card-collection pattern →
V3

By goal (outcome-first)

จัดตาม “อยากพัฒนาทักษะไหน?” — แต่ละทักษะนำด้วยผลลัพธ์ (เช่น “พูดโต้ตอบได้มั่นใจขึ้น”) แล้วค่อยลิสต์แบบฝึก. ตัวเด่น = จุดเริ่มที่แนะนำ.

หลัก: Inspired (outcome-first) + Change by Design →
