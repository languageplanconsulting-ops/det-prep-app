Mock Test · หน้าก่อนเริ่ม + ตอนทำข้อสอบ

หน้าทำ Mock — 4 แบบ (ครบทั้ง flow)

แต่ละแบบมี 2 จอ: ① ก่อนเริ่มสอบ (เลือกชุด · ยืนยันเครดิต · กดเริ่ม) → ② ตอนทำข้อสอบ (timer + โจทย์ + ปุ่มส่ง). เดิมทั้งคู่ยังเป็น brutalist (กรอบดำหนา · 4 กล่องสถิติ · preflight modal ⚠️ + sidebar เย็น). ทั้ง 4 แบบใช้ภาษาออกแบบ soft-modern เดียวกับ revamp ที่ผ่านมา (การ์ดขาวมุมโค้ง · #004AAD/#FFCC00 · ไม่มี gradient) แต่ละแบบนำด้วยหลักจากหนังสือคนละเล่ม

V1

Calm Focus

จอเดียว คอลัมน์เดียว ตัด sidebar/สถิติทิ้ง. แถบบนบาง = ข้อที่เท่าไหร่ + เวลา. ปุ่มส่งเต็มกว้างติดล่าง.

Krug · Fitts · Hick — สมาธิสูงสุด ของรบกวนน้อยสุด

① ก่อนเริ่ม → ② ตอนทำ
V2

Progress Journey

rail ซ้ายเป็นแผนที่ 20 ข้อ จัดกลุ่มตามทักษะ — ทำแล้ว ✓ / ข้อนี้กระพริบ / ข้างหน้า lock. หน้าก่อนเริ่มรองรับ 50+ ชุด: intro ชัด + ระบบแนะนำ 3 ชุด + คลังค้นหา/กรอง.

Tidwell · Nodder · Hick — ความก้าวหน้า ไม่ใช่ความกลัว · เลือกง่ายแม้ชุดเยอะ

① ก่อนเริ่ม → ② ตอนทำ
V3

Coach Companion

มี “พี่ดอย” อยู่ข้าง ๆ — ทิปเฉพาะข้อ + ให้กำลังใจ + แถบ “นี่คือการซ้อม” ลดความกังวลก่อนลงมือ.

Change by Design (เห็นใจ) — อุ่นใจ ลดความเครียดสอบ

① ก่อนเริ่ม → ② ตอนทำ
V4

Exam Room (mobile)

ออกแบบจากมือถือก่อน. status strip บางสุดด้านบน, โจทย์ตรงกลาง, ปุ่มส่งเต็มกว้างติดล่างในเขตนิ้วโป้ง.

Fitts · Krug — เหมือนห้องสอบจริง แต่ผ่อนคลายกว่า

① ก่อนเริ่ม → ② ตอนทำ

ร่วมกันทั้ง 4 แบบ

  • • ไม่มี gradient · ไม่มีคำว่า “AI” · เสียงโค้ช = พี่ดอย แยกจากเสียงทีมขาย (ตามกติกา revamp)
  • • ตัวอย่างใช้ task เดียวกัน (อ่านแล้วเขียน ข้อ 7/20) เพื่อเทียบ layout ได้ตรง ๆ
  • • เลข/เวลาเป็น JetBrains Mono · ตัวอักษร IBM Plex Sans Thai (จริงในแอปคือ Inter)
  • • กล่องท้ายแต่ละแบบอธิบายหลักการ — ลบออกตอนพอร์ตจริง
