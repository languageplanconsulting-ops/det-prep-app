วิธีใช้ Mock Test · How mock tests work

① About the sets

ข้อมูลเกี่ยวกับชุดข้อสอบ

ข้อสอบ Mock Test จะมีการอัปโหลดชุดใหม่ทุกเดือน ตั้งแต่ ม.ค. 2026 — ข้อสอบแต่ละชุดนำมาจากข้อสอบ DET จริงที่ใช้สอบในช่วงเดือนนั้น ๆ ในแต่ละเดือนจะมี 6 ชุด ตรงกับโควต้าที่ได้รับต่อเดือน เลือกเริ่มจากชุดไหนก็ได้ ไม่มีลำดับบังคับ

② Before you start

สิ่งที่ต้องรู้ก่อนเริ่มทำข้อสอบ

ใช้เวลาประมาณ 1 ชั่วโมง – 1 ชั่วโมง 15 นาที ควรจัดเวลาให้พร้อม ทำค้างไว้แล้วออกระบบ = นับว่าใช้โควต้าชุดนั้นไปแล้ว

ตัวเลขชุด (1, 2, 3) ไม่ได้บ่งบอกระดับความยาก — ทุกชุดยากเท่ากัน เพราะเป็นแบบ Adaptive ระบบจะปรับยากขึ้นตามผลที่คุณตอบ ยิ่งตอบถูกมาก ข้อต่อไปยิ่งยาก เจอข้อตอบไม่ได้ ไม่ต้องตกใจ — นั่นคือเป้าหมายของระบบที่ต้องการหาขีดจำกัดของเรา

③ Practice, not the real exam

ข้อสอบนี้คือการฝึก ไม่ใช่สอบจริง

ระหว่างทำ Mock Test ไม่ต้องกังวลเรื่องการตั้งกล้อง จะเปิดอินเทอร์เน็ตค้นข้อมูลก็ได้ — นี่คือการฝึกซ้อม ไม่มีการตรวจจับหรือบันทึก

แต่ในการสอบจริง สิ่งเหล่านี้สำคัญมาก โดยเฉพาะการตั้งกล้องและสภาพแวดล้อม Duolingo กำหนดไว้อย่างเข้มงวด — ควรศึกษากฎของ Duolingo ก่อนสอบจริง

④ Feedback & problems

Feedback และการแจ้งปัญหา

หลังจากทำแต่ละชุดเสร็จ จะได้ Feedback แบบเฉพาะบุคคล บันทึกเก็บไว้ติดตามพัฒนาการได้ — Feedback นี้จะช่วยให้เห็นชัดว่าจุดอ่อนอยู่ที่ไหน นำไปวางแผนการเรียนได้ตรงจุด

พบปัญหาระหว่างทำ แจ้งได้ที่ languageplan.consulting@gmail.com
