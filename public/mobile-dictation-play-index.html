ปรับปุ่ม "เล่นเสียง" + หน้าตาข้อสอบตามคำบอก 🎧

หน้าเดิมมีแค่ปุ่มน้ำเงินแบน ๆ + ช่องพิมพ์ + ปุ่มส่ง ทั้ง 3 แบบทำให้ปุ่มเล่นเสียงเด่นและน่ากด มี "คลื่นเสียง" ขยับตอนเล่น สื่อชัดว่า ฟัง→พิมพ์→ส่ง และบอกว่าฟังซ้ำได้ไม่จำกัด

📱 หน้าจริงที่แก้: DictationRunnerScreen (มือถือ) — ปุ่ม ▶ เล่นเสียง + ช่องพิมพ์ + ส่งคำตอบ · ฟังซ้ำได้ไม่จำกัด (นับจำนวนครั้ง) · คลื่นเสียงขยับจริงตอนเล่น
▶️
VERSION 1

Big Player

ปุ่มเล่นวงกลมใหญ่กลางจอแบบเครื่องเล่นเพลง มีวงกระเพื่อม + คลื่นเสียงขยับ ปุ่ม "ฟังอีกครั้ง" ชัด กดง่ายด้วยนิ้วโป้ง

  • ปุ่มเด่นสุด น่ากดสุด (Fitts's Law)
  • รู้สึกเหมือนแอปฟังเพลง สนุก
  • ช่องพิมพ์อยู่ล่าง โฟกัสที่การฟังก่อน
🃏
VERSION 2

Card Steps

แยกเป็น 2 การ์ดชัดเจน ① ฟังเสียง (ปุ่มเล่น + คลื่น + ปุ่มซ้ำ) ② พิมพ์คำตอบ (ช่อง + นับคำ) ปุ่มส่งเต็มความกว้างล่างสุด

  • เป็นขั้นตอน 1-2 เข้าใจง่ายที่สุด
  • สอดคล้องกับดีไซน์การ์ดทั้งแอป
  • เรียบ สงบ ไม่รก · ปลอดภัยสุด
🎯
VERSION 3

Focus Mode

โซนฟังเป็นพื้นน้ำเงินเข้มเต็มด้านบน ปุ่มเล่นเหลืองใหญ่ + คลื่น สร้างสมาธิ ช่องพิมพ์เป็นเส้นใต้บนพื้นครีมด้านล่าง

  • ดูพรีเมียม โดดเด่น มีสมาธิกับการฟัง
  • แยกโซน "ฟัง" กับ "พิมพ์" ชัดด้วยสี
  • ต่างจากหน้าอื่นในแอปมากสุด

เหมือนกันทั้ง 3 แบบ (สิ่งที่ดีขึ้นจากเดิม)

• ปุ่มเล่นเสียง ใหญ่ขึ้น เด่นขึ้น มีวงกระเพื่อม/หายใจ  •  คลื่นเสียงขยับ ตอนกำลังเล่น (ไม่ต้องเดาว่าเสียงเล่นอยู่ไหม)  •  ปุ่ม "ฟังอีกครั้ง" แยกชัด + บอก "ฟังซ้ำได้ไม่จำกัด · เล่นแล้ว N ครั้ง"  •  ลำดับชัด ฟัง → พิมพ์ → ส่ง  •  ช่วยลดกังวล "ไม่ต้องสนใจตัวพิมพ์ใหญ่-เล็ก"  •  ใช้สี/ฟอนต์แบรนด์เดิม (น้ำเงิน/เหลือง + Prompt)
