เปลี่ยนอะไรจาก Option 1 (ตามหนังสือ Krug + Cagan):

  • English-first — โจทย์อังกฤษเป็นพระเอก (15.5px ตัวหนา) เพราะข้อสอบจริงเป็นอังกฤษ · ไทยเป็น "คำแปล" ตัวเล็กใต้เส้น ช่วยให้เข้าใจ ไม่แย่งสายตา
  • ไม่มี gradient เลย — label เป็นเหลืองทึบ · การ์ด done เป็นเขียวอ่อนทึบ · ปุ่มสีทึบล้วน
  • List แนวตั้งแทน grid (Krug: scan over read) — โจทย์ยาวอ่านง่ายกว่าเมื่อ 1 ข้อ/แถว ไม่บีบ 2 คอลัมน์
  • Status spine ซ้ายมือ — เขียว = ทำแล้ว · เทา = ยังไม่ทำ · รู้สถานะใน 1 วินาทีโดยไม่ต้องอ่าน (Krug: convention)
  • คะแนนเป็นพระเอกของ done state (Cagan: outcome-first) — "138/160" + ปุ่ม "ทำใหม่" ปิด feedback loop "เขียน → ได้คะแนน → ทำให้ดีขึ้น"
  • max-w แคบลง (3xl) — บรรทัดอังกฤษยาวกำลังดี อ่านสบายตา (optimal line length)
