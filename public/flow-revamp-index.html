เส้นทางผู้ใช้ฟรี — 3 เวอร์ชัน

สตอรีบอร์ดว่า ผู้ใช้ฟรีเห็นอะไรในแต่ละจุด — ตั้งแต่ลองฟรี → เข้าสู่ระบบ → สิทธิ์ฟรี → คลังฝึก → ผลวินิจฉัย → ชนกำแพง (ใช้สิทธิ์ครบ) → อัปเกรด → จ่าย PromptPay → สำเร็จ. สไตล์ Clarity Pro · ภาษาไทยนำ · ไม่มีคำว่า "AI" · ไม่ใช้ dark pattern.

จุดที่แก้ (rough edges จากของจริง)

  • • กำแพง login ก่อนได้ลอง — ทำให้เห็นคุณค่าก่อน / sign-in รู้สึกเบาลง
  • • เครดิตฟีดแบ็ก "เลือก 1" ไม่ชัดว่ากดแล้วผูกไหม — ทำให้ชัด
  • • ผลวินิจฉัย กดแล้วเด้งไปหน้าราคาเต็มแบบไม่มีบริบท — ให้ next step ที่ต่อเนื่อง
  • • หน้าบล็อก รู้สึกเหมือนโดนลงโทษ — เปลี่ยนเป็นชวนต่อแบบสุภาพ
  • • PromptPay async ไม่รู้ว่าจ่ายสำเร็จไหม — เพิ่มสถานะ "กำลังยืนยัน" + ยืนยันชัด
V1

Gentle — เรียบง่าย ไม่กดดัน

ทุกหน้าจอเข้าใจทันทีว่าอยู่ตรงไหน เกิดอะไรขึ้น และต้องทำอะไรต่อ (ปุ่มเดียว) · บล็อก = "คุณลองฟรีครบแล้ว 🎉" ไม่ใช่การลงโทษ.

หลัก: Don't Make Me Think + The Mom Test →
V2

Show the value — เห็นคุณค่าก่อน

ทุกจุดที่ขอให้สมัคร/จ่าย จะ โชว์ สิ่งที่จะปลดล็อก (ตัวอย่างเบลอ ๆ ให้เห็น), ความคืบหน้า และรีวิวจริง — โน้มน้าวแบบมีจรรยาบรรณ ไม่หลอก.

หลัก: Designing Interfaces + ethical persuasion →
V3

Momentum — โฟกัสเป้าหมาย

ทุกก้าวตอกย้ำเป้าหมายคะแนนและสิ่งที่ลงแรงไปแล้ว — การฝึกต่อ/อัปเกรดเลยรู้สึกเหมือน "ไปต่อให้ถึงเป้า" ไม่ใช่ธุรกรรม · มี progress tracker ตลอดเส้นทาง.

หลัก: Change by Design (empathy) + goal-gradient →
