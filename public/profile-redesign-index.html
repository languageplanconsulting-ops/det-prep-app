หน้า Profile — 3 เวอร์ชันให้เลือก

ลูกค้าบอกว่าหน้าเดิม “งง อ่านไม่รู้เรื่อง” เพราะมันเป็นหน้าคิดบิล/โควต้า 8 การ์ดที่หน้าตาเหมือนกันหมด, แยก Literacy เป็น 3 การ์ด (เหมือนมีสิทธิ์ 3 เท่า), แถบ % เดินสวนทางกับคำว่า “เหลือ”, และไม่มีความก้าวหน้าของผู้เรียนเลย ทั้ง 3 เวอร์ชันนี้แก้ปัญหาเดียวกัน แต่คนละสไตล์

เหมาะถ้า: อยากผลักการเติมเครดิต/อัปเกรด (รายได้)

ดูเวอร์ชันนี้ →

ปัญหา 10 ข้อของหน้าเดิม → แก้ยังไงในทั้ง 3 เวอร์ชัน

ปัญหาเดิมวิธีแก้
1. เป็นหน้าคิดบิล ไม่ใช่โปรไฟล์V1 เพิ่มความก้าวหน้าผู้เรียนขึ้นก่อน · V2/V3 เริ่มด้วยชื่อ+แพ็กเกจบรรทัดเดียว
2. Literacy แยก 3 การ์ด (ตัวเลขเดียวกัน)รวมเป็นก้อนเดียว มีคำอธิบายว่าใช้ร่วมกันสำหรับ 3 ข้อสอบ
3. 8 การ์ดหน้าเหมือนกัน อ่านยากลดเหลือรายการ/แถวเรียบ ๆ + จัดกลุ่ม + ดัน 2 สิทธิ์สำคัญให้เด่น
4. แถบ % เดินสวนคำว่า “เหลือ”แถบเต็มขึ้นตามการใช้งาน + ป้าย “เหลือ X / total” ตรงไปตรงมา
5. สีแดงทำผู้ใช้ฟรีตกใจใช้สีสงบ (เขียว/เหลือง/น้ำเงิน) สื่อสถานะ ไม่ใช่สัญญาณเตือนภัย
6. มีแผง “วิธีอ่านหน้านี้” = หน้าซับซ้อนเกินเอาออก/ย่อเป็น “ดูเพิ่ม” เพราะหน้าเข้าใจได้เองแล้ว
7. สถิติที่ 3 “ตรวจได้ 8 รายการ” ไร้ความหมายแทนด้วยวันเหลือ / คะแนน Mock / XP ที่ผู้ใช้สนใจจริง
8. ไม่มีความก้าวหน้าการเรียนเลยV1 เพิ่ม streak + Mock ล่าสุด + จุดที่ควรฝึกต่อ
9. ไม่มีปุ่มจัดการบัญชีเพิ่ม จัดการแพ็กเกจ · เติมเครดิต · เปลี่ยนรหัส · ออกจากระบบ
10. ชื่อแพ็กเกจซ้ำ 3 ที่ + error เงียบโชว์แพ็กเกจครั้งเดียว (เวอร์ชันจริงจะเพิ่มแจ้งเตือนเมื่อโหลดยอดไม่สำเร็จ)
