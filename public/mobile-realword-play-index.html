เล่น "คำศัพท์จริง" — 5 แนวทางออกแบบ 🎯

นี่คือหน้าจอ ตอนเล่นจริง — แตะเลือกคำที่เป็นภาษาอังกฤษจริงจากตาราง แล้วกดตรวจ ปัญหาเดิม: ไม่รู้ว่าเลือกไปกี่คำ, สถานะ "เลือกแล้ว" ไม่ชัด, ไม่บอกว่า "คำมั่ว" คืออะไร และไม่บอกว่าเดาผิดโดนหักคะแนน ทั้ง 5 แบบแก้คนละมุมตามหนังสือ

📱 ต้นฉบับที่แก้: RealWordRunnerScreen (มือถือ) — ตาราง 18 ช่อง (คำจริง+คำมั่ว) แตะเลือก · คะแนน = ถูก − ผิด · ตรวจแล้วไฮไลต์ เขียว✓/แดง✕/เหลืองพลาด

🔗 ก่อนหน้านี้: หน้า เลือกชุด (set-picker) มี 5 เวอร์ชันแยกที่ mobile-realword-index.html

👆
VERSION 1
Don't Make Me Think — Krug

ชัดจนไม่ต้องคิด ปลอดภัยสุด

โจทย์ตัวใหญ่อ่านง่าย + ตัวอย่าง "คำมั่ว" ชัด ๆ · ช่องที่เลือกเป็นสีน้ำเงิน + ✓ ไม่มีทางเข้าใจผิด · ตัวนับ "เลือกแล้ว 5 คำ" ลอยเหนือปุ่มเดียว

  • แก้ทุกความกำกวมของหน้าจอเดิม
  • ตาราง 2 คอลัมน์ ช่องใหญ่ กดง่ายด้วยนิ้ว
  • ทำเป็น React Native ได้เร็วที่สุด
🚦
VERSION 2
The Design of Everyday Things — Norman

ฟีดแบ็ก + กติกาชัด

บอก "กติกาคะแนน" ล่วงหน้า (ถูก +1 · เดาผิด −1) · หน้าตรวจไฮไลต์ทุกคำ เขียว✓ / แดง✕ / เหลือง "พลาด" พร้อมคำอธิบายสีครบ

  • Feedback: เห็นชัดว่าเลือกถูก/ผิด/พลาดตรงไหน
  • Mapping: รู้ผลของการเดาผิดตั้งแต่ก่อนกด
  • *ตัวอย่างนี้โชว์หน้าจอ "หลังตรวจ"
☑️
VERSION 3
Designing Interfaces — Tidwell

ตารางคัดเลือก (Grid Select)

ใช้แพตเทิร์นเลือกหลายช่องมาตรฐาน — มีช่องติ๊ก ✓ หน้าคำ + แถบเครื่องมือบอกจำนวน + ปุ่ม "ล้าง" + แถบความคืบหน้า เป็นระเบียบ สแกนเร็ว

  • เป็นแพตเทิร์นที่ผู้ใช้คุ้นเคย เข้าใจทันที
  • ช่องติ๊กสื่อ "เลือกได้หลายอัน" ชัดเจน
  • มีปุ่มล้าง/ตัวนับ ควบคุมง่าย
🧑‍🏫
VERSION 4
Change by Design — Brown

โค้ชพี่ดอย (ลดความกลัว)

พี่ดอยให้เคล็ดลับ ("คำจริงมักอ่านลื่น") + คำปลอบ "ไม่ชัวร์ก็ข้ามได้ เดาผิดไม่หัก XP" ลดความกดดัน ช่องเลือกเป็นดาวเหลืองอบอุ่น

  • สอนวิธีคิดระหว่างเล่น ไม่ใช่แค่ให้ทำ
  • ลดความกลัวผิดของผู้เรียนไทย
  • เชื่อมแบรนด์ "พี่ดอย" ที่มีอยู่แล้ว
🔥
VERSION 5
Evil by Design — Nodder

เกมจับผิดคำ (มีจังหวะ)

ทำให้เป็นเกม — HUD บอก "จับได้ 5/9 คำ" + คอมโบ ×3 + แถบเวลา + ช่อง "+1" เด้ง กระตุ้นให้แตะไว จับครบรับโบนัส (ตัวเลขจริง ไม่หลอก)

  • เอนเกจสูงสุด · เชื่อมระบบ XP/สตรีค
  • เวลา = ความตื่นเต้น (เลือกออปชันเปิด/ปิดได้)
  • เสี่ยงกดดันผู้เรียนบางคน · ต้องปรับพอดี

แก่นที่ทั้ง 5 แบบแก้เหมือนกัน

• ตัวนับสด "เลือกแล้ว X คำ" ตลอดเวลา (เดิมไม่มี)  •  สถานะ "เลือกแล้ว" ชัดเจน (สี + ✓/⭐/ช่องติ๊ก)  •  อธิบาย "คำมั่ว" เป็นรูปธรรม พร้อมตัวอย่าง  •  สื่อสาร กติกาหักคะแนนเมื่อเดาผิด อย่างซื่อสัตย์ (เดิมซ่อนไว้)  •  ปุ่มตรวจ บอกจำนวนที่เลือก ก่อนกด

ต่างกันที่โทน: V1 เรียบ-ตรงประเด็น · V2 เป็นระบบ-ให้ฟีดแบ็ก · V3 เป็นระเบียบ-คุ้นเคย · V4 อบอุ่น-ให้กำลังใจ · V5 สนุก-เร้าใจ — เลือกได้ตามความรู้สึกที่อยากให้ผู้เรียนได้ แล้วผมพอร์ตลง RealWordRunnerScreen.tsx ให้จริง (admin-gated ก่อน) · ถ้าอยากให้ธีมเข้าชุดกัน แนะนำจับคู่กับเวอร์ชันที่เลือกไว้ในหน้า set-picker
