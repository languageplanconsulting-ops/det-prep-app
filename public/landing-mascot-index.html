VERSION 1

Hero Buddy ปลอดภัยสุด

มาสคอตเป็น "เพื่อนคู่คิด" ยืนข้างพาดหัวใน Hero — ท่าคิดของมาสคอตเชื่อมกับสาร "หาจุดอ่อน" พอดี มีลูกโป่งความคิด "อ่อนตรงไหนนะ?" แล้วโผล่อีกทีตอนโน้ตฟีดแบ็ก + ปุ่มสุดท้าย

  • สะอาด สงบ ตรงกับโทน Calm Clarity เดิม
  • เปลี่ยนน้อย เสี่ยงน้อย พอร์ตง่ายสุด
  • ให้ความรู้สึกอบอุ่น เข้าถึงง่าย
💬
VERSION 2

Coach (พูดนำทาง)

มาสคอต (ตั้งชื่อชั่วคราวว่า "น้องแพลน") เป็นโค้ชที่พูดนำทางทั้งหน้า — Hero เป็นลูกโป่งคำพูดบอก value prop, แต่ละหัวข้อมีมาสคอตพูดเกริ่นสั้น ๆ โทนครีมอบอุ่น

  • มีบุคลิก/เสียงชัด สร้างความผูกพันกับแบรนด์
  • ลดความกลัวของผู้เรียน (empathy-first)
  • ผูกกับ "น้องแพลน" เป็นตัวละครประจำแอปได้
✨
VERSION 3

Playful (โผล่ทั้งหน้า)

มาสคอตเป็นตัวการ์ตูนสนุก ๆ ที่โผล่ทั่วหน้า — เด้งใน Hero พร้อมสติกเกอร์ "+25 คะแนน 🎉", โผล่ขอบการ์ด, รีแอกต์ข้างผลสอบ พื้นจุด ๆ สดใส สไตล์ Gen-Z

  • สะดุดตา จำง่าย เอนเกจสูง
  • เหมาะกลุ่มนักเรียน/วัยรุ่น
  • เสี่ยงดู "เล่น" เกินถ้าใช้เยอะไป · ต้องคุมพอดี

เหมือนกันทั้ง 3 แบบ

• ใช้ มาสคอตตัวเดียวกัน (public/mascot-doy.png พื้นหลังโปร่ง) ทั้งหน้า  •  คงพาดหัว/สาร/ปุ่มจริงจากหน้า Calm Clarity ที่ใช้อยู่  •  วางมาสคอตใน Hero + โน้ตฟีดแบ็ก + ปุ่มสุดท้าย เป็นอย่างน้อย  •  ไม่มีคำว่า "AI" — ฟีดแบ็กคือ "ระบบ/ฐานข้อมูล 6 ปีของ English Plan"

ต่างกันที่ บทบาทของมาสคอต: V1 เพื่อนคู่คิด (นิ่ง สวย) · V2 โค้ชที่พูดนำ (มีเสียง อบอุ่น) · V3 ตัวการ์ตูนสนุก (สติกเกอร์ เด้ง)

หมายเหตุ: ชื่อ "น้องแพลน" ใน V2 เป็นชื่อตั้งชั่วคราว — บอกได้ว่าอยากใช้ชื่ออะไร  ·  มาสคอตปัจจุบันเป็นภาพครึ่งตัว/หน้า ถ้าจะใช้จริงควรมีเวอร์ชัน เต็มตัว + หลายท่าทาง (ยิ้ม/ชูนิ้ว/ดีใจ) เพื่อความหลากหลาย เลือกเวอร์ชันที่ชอบแล้วผมพอร์ตลง AdminLandingPageClient.tsx ให้จริง
