"คำศัพท์จริง" — 5 แนวทางออกแบบใหม่ 📚

หน้าจอเดิมเป็นแค่รายการ "รอบที่ 1 · ชุด N" ที่หน้าตาเหมือนกันหมด — ไม่บอกว่าคุณอยู่ตรงไหน ทำอะไรต่อ หรือทำไปแล้วกี่ชุด แต่ละเวอร์ชันหยิบ แก่นจากหนังสือดีไซน์เล่มหนึ่ง มาแก้ปัญหานี้คนละมุม

📱 ต้นฉบับที่แก้: ExamPracticeScreen (มือถือ) — แท็บระดับ ง่าย/ปานกลาง/ยาก + รายการชุดธรรมดา · ใช้ข้อมูลจริง (ยังไม่ทำ / คะแนน% / ✓ ผ่าน / ★ เต็ม / 🔒 ล็อก)
👆
VERSION 1
Don't Make Me Think — Krug

ทำต่อทันที ปลอดภัยสุด

ปุ่มเดียวเด่นชัด "ทำต่อ ชุดที่ 8" — ระบบเลือกให้ ผู้เรียนไม่ต้องคิดว่าจะกดชุดไหน ที่เหลือเป็นรายการเลือกเองแบบจาง ๆ

  • แก้ "ต้องคิด" → เห็นปุ๊บรู้ปั๊บว่ากดอะไร
  • หนึ่งการกระทำหลัก · สแกนเร็ว โปร่ง
  • ทำเป็น React Native ได้เร็วที่สุด
🎛️
VERSION 2
The Design of Everyday Things — Norman

บอร์ดสถานะ + เป้าแม่นยำ

ทุกชุดส่ง "สัญญาณ" สถานะชัดด้วยสี (ผ่าน/ทำซ้ำ/ยังไม่ทำ/ล็อก) + แถบ mapping ความแม่นจริง → เป้า 95% พร้อมคำอธิบายสีกำกับ

  • Signifiers + Feedback: รู้สถานะทุกชุดในตาเดียว
  • Mapping: เห็นว่าต้องดันอีกเท่าไรถึงปลดล็อก
  • กริดตัวเลขกะทัดรัด เห็นครบ 20 ชุด
🧭
VERSION 3
Designing Interfaces — Tidwell

แผนผังลำดับ (บท)

ใช้แพตเทิร์น "Sequence Map" — จัดชุดเป็น "บท" ต่อกันด้วยเส้นราง มีคำอธิบายว่า รอบ/บท คืออะไร ปลดล็อกไล่ลำดับ

  • อธิบาย "รอบ/ชุดคืออะไร" ที่เดิมไม่เคยบอก
  • เห็นลำดับ+ความคืบหน้าแต่ละบทชัด
  • เป็นแพตเทิร์นมาตรฐานที่ผู้ใช้คุ้นเคย
🧑‍🏫
VERSION 4
Change by Design — Brown

โค้ชพี่ดอย (เห็นใจผู้เรียน)

เริ่มจาก empathy — พี่ดอยพูดให้กำลังใจ ลดความกดดัน เฟรมแต่ละชุดเป็น "ก้าวเล็ก ๆ" ทำผิดได้ โทนอบอุ่น เป็นมนุษย์

  • ลดความวิตกของผู้เรียนไทยที่กลัวภาษาอังกฤษ
  • เชื่อมกับแบรนด์ "พี่ดอย" ที่มีอยู่แล้ว
  • สร้างความรู้สึก "อยากทำต่อ" ด้วยกำลังใจ
🔥
VERSION 5
Evil by Design — Nodder

บันไดพิชิต (เกม แต่ซื่อสัตย์)

ใช้จิตวิทยาจูงใจอย่างมีจริยธรรม — สตรีค 🔥 + เป้า XP รายวัน + บันไดชุด + กล่องรางวัลข้างหน้า กระตุ้น "อีกชุดเดียว" (ตัวเลขจริง ไม่หลอก)

  • เชื่อมระบบ XP/สตรีคที่มี (gamification.ts) โดยตรง
  • Loss-aversion แบบซื่อสัตย์: รักษาสตรีค
  • เอนเกจสูงสุด · เสี่ยงดู "เด็ก/เยอะ" ถ้าเกินพอดี

แก่นที่ทั้ง 5 แบบแก้เหมือนกัน

• เห็น "คุณอยู่ตรงไหน / เหลืออีกกี่ชุด" ทันที  •  แสดง คะแนน% / ✓ ผ่าน / ★ เต็ม จากข้อมูลจริง (`getRealwordProgress`) แทนการ์ดที่เหมือนกันหมด  •  มี ปุ่ม "ทำต่อ" หลักปุ่มเดียว เพื่อไม่ให้เลือกจนงง (Hick's Law)  •  ระดับ ง่าย→ยาก เป็น ขั้นที่ปลดล็อกต่อกัน ไม่ใช่แค่แท็บ

ต่างกันที่ โทน/แนวคิดหลัก: V1 เรียบ-ตรงประเด็น · V2 เป็นระบบ-แม่นยำ · V3 เป็นระเบียบ-อธิบายชัด · V4 อบอุ่น-ให้กำลังใจ · V5 สนุก-เร้าใจ  — เลือกได้ตามว่าอยากให้ผู้เรียนรู้สึกแบบไหน แล้วผมจะพอร์ตลง ExamPracticeScreen.tsx ให้จริง (แบบ admin-gated ก่อน เหมือนทุกครั้ง)
